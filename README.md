# watch

watch is app for viewing videos


## Installation

### Install Electron

```
(local)$ npm i -g electron-prebuilt
```

### Clone & Install NPM Package

```
(local)$ git clone git@github.com:ideyuta/watch.git
(local)$ cd watch
(local)$ npm install
(local)$ npm run g
```


## Develop

### gulp

```
npm run g
```

### electron

```
npm run e
```


## Package Build

```
npm run build
```


## Test

```
(local)$ npm run test
```
